Projeto #1 - Regressão logística utilizando a base de tipos de bolas (prazo: 30/09/2019 às 23:59:59)

  • Tarefa: implementar o método de regressão logística treinado com gradient descent para classificar as imagens de teste da base; para tal, deve-se utilizar apenas NumPy e OpenCV.
  • Entrega: enviar um e-mail para [mauriciops at ufba dot br] com o assunto "MATB16 - Projeto #1" contendo um único anexo chamado 'projeto1.zip'. Esse arquivo deve compactar um diretório chamado 'projeto1' contendo ao menos os seguintes arquivos: 'Dockerfile', 'questionario.pdf', 'requirements.txt', 'test.py', e 'train_and_test.py'. Não altere os arquivos 'Dockerfile' e 'requirements.txt' do exemplo de submissão abaixo. O arquivo 'test.py', quando executado, deve carregar todas as imagens de teste e classificá-las utilizando um modelo pré-gerado em menos de 3 minutos. O arquivo 'train_and_test.py', quando executado, deve carregar todas as imagens de treino e criar um modelo de classificação; depois deve carregar todas as imagens de teste e classificá-las utilizando o modelo gerado; o tempo limite é de 60 minutos. O arquivo 'questionario.pdf' deve ser respondido e gerado a partir do modelo disponibilizado. A base não deve ser colocada dentro do pacote a ser submetido. Siga os exemplos fornecidos para gerar a saída no formato esperado com os parâmetros especificados.
  • Exemplo de submissão: projeto1.zip
  • Como executá-lo?
    1. $ wget https://maups.github.io/disciplinas/labIA/projeto1.zip
    2. $ unzip projeto1.zip
    3. $ cd projeto1
    4. $ sudo docker build -t projeto1 .
    5. $ PATH1=/path/to/dataset/folder/
    6. $ PATH2=/path/to/output/folder/
    7. $ sudo docker run -v $PATH1/:/data:ro -v $PATH2/:/wdata projeto1 bash -c 'python3 test.py /data/base/teste/ /wdata/resultado.txt'
    8. $ sudo docker run -v $PATH1/:/data:ro -v $PATH2/:/wdata projeto1 bash -c 'python3 train_and_test.py /data/base/treino/ /data/base/teste/ /wdata/resultado.txt'
