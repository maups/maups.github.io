  • Corretor [docker]
    • Instruções de uso:
      1. Instale o docker na sua máquina (https://docs.docker.com/install/)
      2. Baixe o corretor
      3. No diretório onde o corretor foi baixado, execute os comandos a seguir
      4. $ unzip pack.zip
      5. $ cd pack
      6. Copie o seu código para o diretório corrente usando o nome "test.cpp" caso tenha feito o trabalho em C ou C++, ou o nome "test.py" caso tenha feito o trabalho em Python2
      7. $ sudo docker build -t compiladores .
      8. Caso seja aluno da Turma 01 com imlpementação em C ou C++, teste sua imlpementação com o seguinte comando:
      9. $ sudo docker run -t compiladores ./checker1 -cpp
      10. Caso seja aluno da Turma 01 com imlpementação em Python2, teste sua imlpementação com o seguinte comando:
      11. $ sudo docker run -t compiladores ./checker1 -py
      12. Caso seja aluno da Turma 02 com imlpementação em C ou C++, teste sua imlpementação com o seguinte comando:
      13. $ sudo docker run -t compiladores ./checker2 -cpp
      14. Caso seja aluno da Turma 02 com imlpementação em Python2, teste sua imlpementação com o seguinte comando:
      15. $ sudo docker run -t compiladores ./checker2 -py
