  • Corretor [docker]
    • Instruções de uso:
      1. Instale o docker na sua máquina (https://docs.docker.com/install/)
      2. Baixe o corretor
      3. Descompacte o pacote do corretor
      4. Copie o seu código para o diretório do corretor usando o nome "test.cpp" caso tenha feito o trabalho em C ou C++, ou o nome "test.py" caso tenha feito o trabalho em Python2 ou Python3
      5. Crie o container do corretor executanto o seguinte comando tendo a pasta do corretor como diretório corrente:

        $ sudo docker build -t compiladores .

      6. Caso seja aluno da Turma 01 com implementação em C ou C++, teste seu trabalho com o seguinte comando:

        $ sudo docker run -t compiladores ./checker1 -cpp

      7. Caso seja aluno da Turma 01 com implementação em Python2, teste seu trabalho com o seguinte comando:

        $ sudo docker run -t compiladores ./checker1 -py2

      8. Caso seja aluno da Turma 01 com implementação em Python3, teste seu trabalho com o seguinte comando:

        $ sudo docker run -t compiladores ./checker1 -py3

      9. Caso seja aluno da Turma 02 com implementação em C ou C++, teste seu trabalho o seguinte comando:

        $ sudo docker run -t compiladores ./checker2 -cpp

      10. Caso seja aluno da Turma 02 com implementação em Python2, teste seu trabalho com o seguinte comando:

        $ sudo docker run -t compiladores ./checker2 -py2

      11. Caso seja aluno da Turma 02 com implementação em Python3, teste seu trabalho com o seguinte comando:

        $ sudo docker run -t compiladores ./checker2 -py3
